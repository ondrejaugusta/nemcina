# 2. Základy podstatných jmen
Podobně jako v češtině rozlišujeme i v němčině tři různé rody – mužský, ženský a střední. Narozdíl od češtiny je v množném čísle jen jeden společný rod.  
Další odlišností němčiny od češtiny je fakt, že se podstatná jména vždy zapisují s velkým písmenem na začátku bez ohledu na to, zda se jedná o vlastní jméno (jako tomu je v češtině).
> [!NOTE]
>
> Používáme členy. V němčině rozlišujeme, podobně jako v angličtině, dva jejich druhy: určitý a neurčitý.
>
> Je dobré se členy učit rovnou se slovy. Časem si pro ně vyvinete cit, ale obzvlášť ze začátku je velmi důležité, abyste se je naučili se slovy dohromady.

## 2.1 Určité členy
Pro **číslo jednotné** určujeme tyto členy:
| Rod mužský | Rod ženský | Rod střední |
| ----------- | ----------- | ----------- |
| der | die | das |

V **čísle množném** používáme jeden souhrnný člen:
| Všechny rody |
| - |
| die |

Podobně jako v angličtině používáme i v němčině určité členy v situacích, kdy je z kontextu zřejmé, o čem konkrétně mluvíme, popřípadě pokud se jedná o nějaký obecně známý koncept.

## 2.2 Neurčité členy
Pro **číslo jednotné** určujeme tyto členy:
| Rod mužský | Rod ženský | Rod střední |
| - | - | - |
| ein | eine | ein |

> [!WARNING]
>
> Pro číslo množné **neexistuje** neurčitý člen. V takovém případě se člen zkrátka vynechá.

## 2.3 Příklady
| Český výraz | Německý výraz s určitým členem | S neurčitým členem |
| - | - | - |
| Muž  | der Herr | ein Herr |
| Žena | die Frau | eine Frau |
| Dítě | das Baby | ein Baby |
| Děti | die Babys | Babys |
