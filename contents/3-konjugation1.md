# 3. Základní pravidla časování sloves v přítomném čase
Pro další část budeme potřebovat mít znalost těchto slov:
- fragen = ptát se,
- lernen = učit se,
- antworten = odpovídat,
- arbeiten = pracovat,
- sein = být,
- haben = mít,
- werden = stávat se, být (v budoucnu).  
&nbsp;  
Zároveň se budeme muset naučit osoby:
| Česky | Německy | Poznámka |
| - | - | - |
| já | ich | |
| ty | du  | |
| on | er  | |
| ona | sie | |
| ono | es | |
| my | wir | |
| vy | ihr | používá se pro oslovení skupiny, jíž tykáme |
| Vy, oni | Sie, sie | pozor na velké písmenko ve vykání! Sie používáme pro oslovení skupiny, ve které jsou členové, kterým vykáme; o jakékoliv skupině pak mluvíme pomocí sie |

## 3.1 Typy časování sloves
Němčina rozlišuje několik typů časování sloves, a s tím i několik typů sloves. Zejména se rozlišují:
- pravidelná (*slabá*) slovesa,
- nepravidelná (*silná*) slovesa,
- pomocná slovesa a
- modální (způsobová slovesa).  
&nbsp;  
V následujícím textu se zaměříme na **slabá** a **pomocná** slovesa.

## 3.2 Časování slabých sloves
Čím více člověk umí německy, tím spíše člověk zjistí, že němčina není prakticky o ničem jiném, než o pravidlech, tabulkách a všemožných poučkách.  
Jinak tomu není ani zde. Pro časování (nejen) slabých sloves si zavedeme následující tabulku koncovek, které budeme přidávat na konce sloves (více v příkladech níže):
| Číslo | Osoba | Zájmeno | Koncovka |
| ----- | ----- | ------- | -------- |
| jedn. | 1.    | ich     | -e       |
| jedn. | 2.    | du      | -st      |
| jedn. | 3.    | er / sie / es | -t |
| množ. | 1.    | wir     | -en      |
| množ. | 2.    | ihr     | -t       |
| množ. | 3.    | Sie / sie | -en    |

Koncovku pak vždy *"přilepíme"* na konec slovesného kmene. Ten zpravidla vytvoříme odtržením infinitivní koncovky *-en* (př. *lernen* → *lern-*).  
Konkrétně pro sloveso *lernen* pak vypadá tabulka časování následovně:
| Osoba č. jedn.   | Tvar   | Osoba č. množ. | Tvar   |
| ---------------- | ------ | -------------- | ------ |
| 1. ich           | lerne  | 1. wir         | lernen |
| 2. du            | lernst | 2. ihr         | lernt  |
| 3. er / sie / es | lernt  | 3. Sie / sie   | lernen |  

Pro sloveso *fragen* analogicky takto:
| Osoba č. jedn.   | Tvar   | Osoba č. množ. | Tvar   |
| ---------------- | ------ | -------------- | ------ |
| 1. ich           | frage  | 1. wir         | fragen |
| 2. du            | fragst | 2. ihr         | fragt  |
| 3. er / sie / es | fragt  | 3. Sie / sie   | fragen |  

Pokud slovesný kmen **končí na -t nebo -d** (jako v případě slovesa *antworten* → *antwort-*), vkládá se mezi něj a koncovku vpona *-e-* všude tam, kde to dává smysl, následovně:
| Osoba č. jedn.   | Tvar   | Osoba č. množ. | Tvar   |
| ---------------- | ------ | -------------- | ------ |
| 1. ich           | antworte  | 1. wir         | antworten |
| 2. du            | antwort**e**st | 2. ihr         | antwort**e**t  |
| 3. er / sie / es | antwort**e**t  | 3. Sie / sie   | antworten | 

Nebo takto v případě slovesa *arbeiten*:
| Osoba č. jedn.   | Tvar   | Osoba č. množ. | Tvar   |
| ---------------- | ------ | -------------- | ------ |
| 1. ich           | arbeite  | 1. wir         | arbeiten |
| 2. du            | arbeit**e**st | 2. ihr         | arbeit**e**t  |
| 3. er / sie / es | arbeit**e**t  | 3. Sie / sie   | arbeiten | 

### 3.2.1 Časování sloves s předponami
V němčině si musíme dát velký pozor na časování sloves s předponami.  
Existují předpony tzv. **odlučitelné**, **neodlučitelné** a **dílem odlučitelné, dílem neodlučitelné** (německy pořadě **trennbares**, **untrennbares**, **teils trennbares, teils untrennbares Präfix**).  
&nbsp;  
Mezi **odlučitelné předpony** se řadí například *ab-*, *auf-*,  *aus-*, *bei-*, *mit-*, *nach-*, *vor-*, *zu-*.  
Mezi **neodlučitelné předpony** řadíme příkladem *be-*, *ge-*, *ent-*, *emp-*, *ver-*, *zer-* a další.  
Do kategorie **dílem odlučitelných, dílem neodlučitelných předpon** (tedy těch, která mají svou odlučitelnost závislou na každém slově) patří ku příkladu *durch-*, *über-*, *um-*, *unter-* apod.  
&nbsp;  
Slovesa s **neodlučitelnými předponami** časujeme klasicky. Vyčasujeme sloveso *beantworten* (= zodpovědět):
| Osoba č. jedn.   | Tvar   | Osoba č. množ. | Tvar   |
| ---------------- | ------ | -------------- | ------ |
| 1. ich           | beantworte  | 1. wir         | beantworten |
| 2. du            | beantwortest | 2. ihr         | beantwortet  |
| 3. er / sie / es | beantwortet  | 3. Sie / sie   | beantworten |

> [!NOTE] Všimněme si
>
> Všimněme si vpony *-e-*, která tentokrát není zvýrazněna, ale vložena byla dle pravidla slovesných kmenů končících na *-t* a *-d*.
>
> Dále na toto nebude upozorňováno.

Zvýšené opatrnosti však musíme dbát u časování sloves s **odlučitelnou předponou**. Předponu pak od slovesného kmene **odtrháváme** a vkládáme **za sloveso**. Níže časováno sloveso *ein|kaufen*, znamenající *nakupovat*:
| Osoba č. jedn.   | Tvar   | Osoba č. množ. | Tvar   |
| ---------------- | ------ | -------------- | ------ |
| 1. ich           | kaufe **ein**  | 1. wir         | kaufen **ein** |
| 2. du            | kaufst **ein** | 2. ihr         | kauft **ein**  |
| 3. er / sie / es | kauft **ein**  | 3. Sie / sie   | kaufen **ein** |

> [!NOTE] Zápis
>
> Často bude používán zápis *[předpona]|[zbytek]* pro slova s odlučitelnou předponou, aby bylo jasné, že se o taková jedná (*ein|kaufen*, *aus|lösen* = něco vyvolat, spustit, apod.).

Vě větě pak odtrhnutou předponu umisťujeme na její samotný konec.  
Věta *Rád nakupuji.* (rád = gern, gerne) se pak přeloží jako *Ich kaufe gerne **ein***.  
&nbsp;  

**Dílem odlučitelné, dílem neodlučitelné předpony** se pak umisťují v závislosti na tom, zda jsou odlučitelné či neodlučitelné.

## 3.3 Časování pomocných sloves
> [!WARNING]
>
> Toto je pro jakékoliv použití němčiny naprosto klíčová část.

Pomocná slovesa budou naší první kategorií, která se nebude řídit klasickými pravidly časování. Nejlépe demonstrováno to je na příkladu.  
&nbsp;  
Časování slovesa *sein* (= být):
| Osoba č. jedn.   | Tvar   | Osoba č. množ. | Tvar   |
| ---------------- | ------ | -------------- | ------ |
| 1. ich           | bin  | 1. wir         | sind |
| 2. du            | bist | 2. ihr         | seid  |
| 3. er / sie / es | ist  | 3. Sie / sie   | sind |

Pozorný čtenář si všimne, že se toto časování vymyká všem doposud zavedeným pravidlům. Nechť je takový čtenář uklidněn tím, že takto odlišná od pravidel němčina často nebývá a toto je jedna z mála výjimek.  
&nbsp;  

Časování slovesa *haben* (= mít):
| Osoba č. jedn.   | Tvar   | Osoba č. množ. | Tvar   |
| ---------------- | ------ | -------------- | ------ |
| 1. ich           | habe  | 1. wir         | haben |
| 2. du            | hast | 2. ihr         | habt  |
| 3. er / sie / es | hat  | 3. Sie / sie   | haben |

Zde lze zpozorovat jakési lehké držení se pravidel, ale lepší bude si tuto tabulku zapamatovat stejně tak, jako tabulku předešlou a následující.  
&nbsp;  

Časování slovesa *werden* (= stávat se, být v budoucnu):
| Osoba č. jedn.   | Tvar   | Osoba č. množ. | Tvar   |
| ---------------- | ------ | -------------- | ------ |
| 1. ich           | werde  | 1. wir         | werden |
| 2. du            | w**i**rst | 2. ihr         | werdet  |
| 3. er / sie / es | w**i**r**d**  | 3. Sie / sie   | werden |

Zde pozor na změnu hlásky *e* na *i* ve druhé a třetí osobě čísla jednotného. Pozor také na atypickou koncovku *-d* ve třetí osobě čísla jednotného, nahrazující klasickou *-t*.

> [!NOTE]
>
> Bohužel je to s němčinou jako s jakýmkoliv jiným jazykem — pravidla je třeba se učit nazpamět a je třeba je neustále procvičovat.
>
> Chtěl bych trochu motivovat čtenáře: zdaleka nejefektivnější metodou učení je mluvení. Nezáleží na tom, jak dobře / špatně zníte (nebo si myslíte, že zníte, realita je často o poznání lepší), hlavní je, že mluvíte. S plyšákem, se zdí, s kamarádem, s rodiči.
>
> Jen mluvením si pravidla osvojíte. Tak se nebojte a mluvte. :)
